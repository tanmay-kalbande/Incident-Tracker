Welcome to the Incident Tracker

Welcome to the Incident Tracker. This tool helps you record, track, and manage incidents efficiently. Please follow the instructions below to use the system effectively.

How to Use

  • To add a new incident, fill out the form under "New Incident" and click "Add Incident".
  • You can search for incidents using various filters in the "Search Incidents" section.
  • Use the pagination buttons to navigate through the incident history.
  • To export or import incidents, use the options in the "Data Management" section.
